# PSAK115 FORENSIC | Beneish Map
Abbrev: PO=performance obligation; TP=transaction price; SSP=stand-alone selling price; OCF=operating cash flow; G/S=goods or services

RATIO→PARA: DSRI[¶9e,15,38] GMI[¶50-58] AQI[¶95,98] TATA[¶35,44] LVGI[¶60-63] SGI[¶9,57] DEPI[¶95b,98] SGAI[¶91,98a]

# S1 IDENTIFY CONTRACT [¶9-16] — DSRI
¶9 valid IF ALL: committed; rights identifiable; payment terms identifiable; commercial substance; ⚠probable collection[ability+intent only; amount may<contract price if variable ¶52]
¶13: no reassess unless significant change (eg. credit deterioration)
¶15-16: ¶9 unmet + consideration received = LIABILITY; release to revenue only if: (a)no remaining obligation+nonrefundable OR (b)terminated+nonrefundable
⚠DSRI: receivables from ¶9e failure; premature release from deferred = channel stuffing

# S2 IDENTIFY PO [¶22-30] — deferred revenue
¶22: PO = distinct G/S OR series of distinct G/S [same transfer pattern + same progress method]
¶27: DISTINCT IF BOTH: (a)customer benefits standalone or with available resources (b)separately identifiable from other promises in contract
¶29: NOT separately identifiable if: (a)significant integration → combined output (b)significant mutual modification (c)high interdependence → cannot fulfill independently
¶30: non-distinct → combine until distinct bundle; may yield single PO
⚠over-split = front load; under-split = defer; either = timing manipulation

# S3 TRANSACTION PRICE [¶46-65] — GMI, LVGI
¶47: TP = expected entitlement, excl. third-party amounts (eg. tax)
¶50-51: variable consideration = discounts; rebates; refunds; credits; concessions; bonuses; penalties; future contingencies
¶53: estimate by: (a)expected value [probability-weighted; large portfolio] OR (b)most likely [two-outcome contracts]
¶55: refund liability = received − probable entitlement; remeasure each period-end
¶56-57 CONSTRAINT: include variable only if highly probable no significant cumulative reversal; reversal risk ↑ if: market/3P/weather exposure; long uncertainty horizon; limited history; wide concession practice; wide outcome range
¶60-63: adjust TP for TVM if significant financing benefit; exempt if gap ≤ 1yr [¶63]
⚠GMI: revenue recognized without ¶56 constraint; LVGI: financing component unrecognized

# S4 ALLOCATE TP [¶73-86] — multi-element
¶74: allocate by relative SSP
¶77: best SSP = observable separate price; contract/list price ≠ SSP
¶79: SSP unobservable → (a)adjusted market assessment (b)cost+margin (c)residual [only if: highly variable price OR never sold standalone]
¶81-83: discount → allocate proportionally unless observable evidence belongs to specific PO
¶85: variable amount → allocate entirely to specific PO if: specifically related to PO effort + consistent with ¶73
⚠incorrect SSP in bundled → front load delivered component → DSRI ↑; receivables inflated

# S5 SATISFY PO [¶31-45] — DSRI, TATA
¶31: recognize revenue WHEN/AS control transfers to customer
¶35 over time IF ANY ONE: (a)simultaneous receive+consume (b)customer controls asset as created (c)no alternative use+enforceable right to payment to date
¶38 point in time indicators: (a)right to payment now (b)legal title transferred (c)physical possession transferred (d)risks+rewards transferred (e)customer accepted
¶44-45: over time only if progress measurable; if not → revenue = recoverable costs only
⚠DSRI ↑: ¶38 criteria unmet but revenue recognized; or ¶9e collectibility failure
⚠TATA ↑: ¶35 unjustified; accrued contract revenue >> actual OCF

# BALANCE SHEET [¶105-109] — deferred revenue flags
¶105: contract liability[consideration received, PO unfulfilled] | contract asset[performed, right conditional] | receivable[unconditional = time only]
⚠contract liability ↓ abnormal = aggressive recognition; contract asset ↑ abnormal = over-accrual; receivable ↑ = DSRI ↑
¶107: contract asset → ECL per PSAK109

# CONTRACT COSTS [¶91-98] — AQI, SGAI, DEPI
¶91-92: capitalize incremental acquisition costs (eg. commission) if recoverable; non-incremental → expense regardless
¶94: practical expedient: expense if amortization ≤ 1yr
¶95: capitalize fulfillment costs ONLY IF ALL: (a)directly related to specific/anticipated contract (b)enhances resources for FUTURE PO (c)expected recoverable
¶98 MUST EXPENSE: (a)G&A [unless explicitly chargeable to customer] (b)waste — materials/labor/resources (c)costs for already-fulfilled PO (d)indistinguishable fulfilled vs unfulfilled
⚠AQI ↑: ¶98 costs capitalized → non-current assets inflated; periodic expense understated
⚠SGAI: SG&A ↓ via improper ¶91 capitalization masks efficiency decline
⚠DEPI: costs amortized slower than economic reality → depreciation charge understated
